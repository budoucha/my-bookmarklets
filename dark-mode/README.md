# Dark Mode Toggle

Toggle dark mode on any website using CSS filters.

## Usage

Click this bookmarklet while on any webpage to toggle dark mode. Click it again to turn dark mode off.

## How it works

The bookmarklet applies CSS `invert` and `hue-rotate` filters to the `<html>` element, effectively inverting all colors while correcting the hue shift. Images, videos, canvases, and SVGs are re-inverted so they keep their original appearance.
